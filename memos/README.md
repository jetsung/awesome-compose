> [Memos][1] 是一个现代、开源、自托管的知识管理和笔记平台，专为注重隐私的用户和组织而设计。Memos 提供了一个轻量级但功能强大的解决方案，用于捕获、组织和共享想法，具有全面的 Markdown 支持和跨平台可访问性。
