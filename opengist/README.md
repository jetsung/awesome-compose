> [opengist][1] 是一个由 Git 提供支持的自托管粘贴。所有代码片段都存储在 Git 存储库中，可以使用标准 Git 命令或 Web 界面读取和/或修改。它类似于 [GitHub Gist](https://gist.github.com/)，但开源并且可以自托管。
