> [gotify][1] 是一个简单的发送和接收消息的服务器用于（通过 WebSocket 实时）发送和接收消息。为此目的，当时开源项目并不多，而且大多数现有项目已无人维护。另外，该服务器还需满足可自行托管这一要求。我们知道市面上有许多免费和商业化的推送服务。
