# alist

> [alist][1] 是一个支持多种存储，支持网页浏览和 WebDAV 的文件列表程序，由 gin 和 Solidjs 驱动。

[1]:https://alist.nn.ci/
[2]:https://github.com/alist-org/alist

## 配置

- 后台->设置->全局->Ocr 接口。设置为以下网址

```
http://ocr:9898/ocr/file/json
```
