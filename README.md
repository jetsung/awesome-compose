## 提交与格式化

- 预提交工具 `pre-commit`  
  https://github.com/pre-commit/pre-commit  
  https://pre-commit.com/

  ```python
  pip install pre-commit

  # ever commit
  pre-commit install

  # format all files
  pre-commit run --all-files

  # format hook id
  pre-commit run <HOOK_ID>
  ```

- 格式化 `yml` 文件  
  https://github.com/jumanjihouse/pre-commit-hook-yamlfmt
