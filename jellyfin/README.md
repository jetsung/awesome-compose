> [Jellyfin][1] 是一款自由软件媒体系统，让您能够掌控媒体的管理与流传输。它是专有软件 Emby 和 Plex 的替代品，可通过多个应用程序，从专用服务器向终端用户设备提供媒体内容。Jellyfin 源自 Emby 的 3.5.2 版本，并移植到了 .NET 平台，以实现全面的跨平台支持。
