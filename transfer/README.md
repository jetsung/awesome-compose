> [transfer.sh][1] 从命令行共享简单快速的文件。此代码包含服务器，其中包含创建自己实例所需的一切。当前支持S3（Amazon S3），GDRIVE（Google Drive），Storj（Storj）提供商和本地文件系统（本地）。
