> [Swagger][1] 是一个流行的开源框架，用于生成、描述、调用和发现 RESTful Web 服务。它能够自动生成 API 文档，方便开发者快速了解和使用 API。同时，Swagger 还支持 API 测试等功能，有助于提高开发效率和 API 质量。
