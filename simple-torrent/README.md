> [Cloud Torrent][1] 是一款用 Go 语言编写的自托管远程种子客户端。用户可以通过远程方式启动种子，这些种子会被下载为服务器本地磁盘上的一组文件，之后这些文件可以通过 HTTP 协议进行检索或流式传输。
